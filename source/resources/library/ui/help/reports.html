Generating Reports
    Setting Fine Rates
    1. Select the Printable Reports tab
    2. Set the View to Fines
    3. Select a Fine Rate
    4. Select a Fine Limit
    5. If the view does not automatically update, select Refresh
    Generating and Printing a Report
    1. Select the Printable Reports tab
    2. Select the View
    3. Select the Sort (If the view isn't fines)
    4. Press "Print" to print using the system Print dialogue
    Viewing Book Status Reports
    1. Select the Book Status tab
    2. The book status report can be seen in the pane
