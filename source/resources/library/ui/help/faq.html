Frequently Asked Questions

Where are Patron Types found?
Patron types can be found inside of the Patron menu. They can be accessed by selecting the Patron Type icon ().
Is sample data available?
Yes. Sample data can be loaded through File > Load Sample Data.
How does the automatic backup work?
The automatic backup feature creates backups of all data files when data is modified and the user saves. This results in additional files being created inside the data directory. In the case that recovery is needed, an IT specialist is able to find the backup file with the data intact, and move them into a new directory and removing the suffixes from the end of the file name.
Help! I cannot modify transactions!
Transactions are unable to modified within the program to ensure integrity of the data. Transactions exist to maintain a record of all checked out and returned books within the system. Transactions may only be viewed within BookKeeper.
Are graphs available in the reports view?
Yes. There are many different graphs included, such as the Book Status and Checkouts over Time tabs, which include a pie chart of book statuses and a chart of book status over time.
How can I access the interactive help (spotlight help)?
Interactive help, also known as spotlight help, can be accessed through selecting the question mark icon () in the top right of the view.
Is it possible to disable animations?
Yes. Animations can be disabled through Preferences > Enable Transitions in the file menu.
Is it possible to change the school name?
Yes. The school name can be changed through Preferences > School Name in the file menu. This change will be saved along with the data files.
Where are data files stored?
Data files, including preferences can be saved into any directory. Using File > Save As... in the file menu, any location can be used to store data. Automatic backups are generated of the data.
What technologies were used?
Technologies used can be found in the README.html file
Where are preferences store?
Preferences are stored in the same directory as the data files are. They are saved with the file name preferences.txt
What format does the data file use?
The data file is based off a modified comma-separated-values (CSV) format. It separates different library data types using constant string separators.
Is BookKeeper cross-platform?
BookKeeper can be run on Windows, MacOS, Linux, and any other operating system with an available JVM.
Where can I find the source code?
The source code may be found within the source folder located at the root of the submission or online at Github.
