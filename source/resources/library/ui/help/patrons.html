Managing Patrons
  • Add a Patron
    1. Press "New Patron"
    2. Fill out First Name and Last Name
    3. Select a Patron Type from the dropdown menu
    4. Press "Update"
  • Delete a Patron
    1. Search for the Patron using the search bar
    2. Select the Patron you wish to delete from the table
    3. Press "Delete"
  • Updating a Patron
    1. Search for the Patron using the Filter box
    2. Select the Patron row from the Table
    3. Update the fields you wish to modify
    4. Press "Update"
  • Managing Patron Types
    • Panel for managing Patron types. Patron types can be applied to Patrons to set checkout limits
    • Functions: Add type, Delete type, Edit type, View type Limits
    • Select the Patron Types Icon ()
    • Or Select the hyperlink on "Patron Types" next to the dropdown menu
  • To return Home, press the home icon to the right of the search bar
