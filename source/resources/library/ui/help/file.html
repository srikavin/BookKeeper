Navigating the File Menu

File

  • New library data files can be created using New
  • Open library data using Open
  • Save the current library to the file using Save
    • Previous data files are automatically backed up whenever saving
  • Save the current library to a new file using Save As...
  • Sample data can be loaded to quickly demo the application
  • Quit the application using Quit

Preferences

  • Disable Animations by unchecking Enable Animations

Help

  • View Documentation by clicking Documentation
  • View information about this program using About
