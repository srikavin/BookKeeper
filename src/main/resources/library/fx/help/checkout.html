Checkout and Return
  • Checkout a Book
    1. If you know the Patron ID, type it into the "Patron ID" field and press Enter
    2. Otherwise, press the search icon inside the text field
    3. Search for the patron using the popup Window
    4. Highlight the patron and press "Select"
    5. Repeat this process for the "Book ID" field
    6. Press the "Checkout" Button to checkout the book for the Patron
    7. If the selected book is already checked-out, the librarian can override it at their discretion
  • Return a Book
    1. If you know the Patron ID, type it into the "Patron ID" field and press Enter
    2. Otherwise, press the search icon inside the text field
    3. Search for the patron using the popup Window
    4. Highlight the patron and press "Select"
    5. Use the Filter box at the top right to search for the book to return
    6. Select the book in the table
    7. Press the "Return" button
