Managing Patron Types
  • Add a Patron Type
    1. Press "New Patron Type"
    2. Fill out Name and set the Checkout Days and Limit
    3. Press "Update"
  • Delete a Patron Type
    1. Search for the Patron Type using the search bar
    2. Select the Patron Type you wish to delete from the table
    3. Press "Delete"
  • Updating a Patron Type
    1. Search for the Patron Type using the Filter box
    2. Select the Patron Type row from the Table
    3. Update the fields you wish to modify
    4. Press "Update"
  • To return Home, press the home icon to the right of the search bar
