Managing Books
  • Add a Book
    1. Press "New Book"
    2. Fill out all editable Fields
    3. Press "Update"
  • Delete a Book
    1. Search for the Book using the search bar
    2. Select the book you wish to delete from the table
    3. Press "Delete"
  • Updating a Book's Details
    1. Search for the Book using the Filter box
    2. Select the Book row from the Table
    3. Update the fields you wish to modify
    4. Press "Update"
  • To return Home, press the home icon to the right of the search bar
