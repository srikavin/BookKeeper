Navigating the Main Menu

  1. Patrons
    • Panel for managing Patrons in the BookKeeper System
    • Functions: Add Patron, Delete Patron, Edit Patron, View Patron Information
  2. Types
    • Panel for managing Patron types. Patron types can be applied to Patrons to set checkout limits
    • Functions: Add type, Delete type, Edit type, View type Limits
  3. Books
    • Panel for managing Books in the BookKeeper Library
    • Functions: Add book, Remove book, Edit book, View book Information
  4. Reports
    • Panel for generating and printing reports
    • Functions: Generate reports, Print reports, View reports
  5. Checkout and Return
    • Panel for Checking out and Returning Books to the Library
    • Functions: Check out return books for Patrons
